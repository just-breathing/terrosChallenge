- chess game with 8x8 grid 
- implement how each piece moves


### Implementation 
> [!NOTE] used Vite with tailwind css

## Icons
- r -> rook
- k -> Knight
- kg -> King
- q -> Queen
- b -> bishop
- p -> Pawn


## run this project 
 type ```npm run dev``` on terminal
-  dev server is on port 3000 
 ```  http://localhost:3000```
